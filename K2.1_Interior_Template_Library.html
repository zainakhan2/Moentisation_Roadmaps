SPECS

📐 KDP Print Specifications — Master These First

SpecificationYour BooksDetails
Trim sizes6×9" (journals) and 8.5×11" (planners/workbooks)6×9 = portable, fits in a bag, standard journal size. 8.5×11 = large workspace, better for planners with tables and trackers.
BleedNO BLEED for all 8 booksNone of your books have images running to the page edge. Lines, boxes, and text stay within margins. No-bleed = simpler setup, no dimension adjustments needed.
PaperWhite paper, black & white inkCheapest print cost = highest royalty. Cream paper adds warmth but costs more. White + B&W is the standard for journals/planners.
BindingPaperbackLowest cost, fastest to publish. Hardcover is an upgrade for later once you validate demand.
Page count100–130 pages per bookMinimum: 24 pages. Sweet spot for journals/planners: 100–130 pages. Enough content to feel substantial, low enough print cost to maintain good royalties at $9.99–$12.99.

Margin6×9" books8.5×11" booksWhy

Bottom0.5"0.5"Space for page numbers/footers
Outside (right on odd, left on even)0.5"0.5"Thumb space for the reader
Inside/Gutter0.625" (100 pages) — 0.75" (130+ pages)0.625" (100 pages) — 0.75" (130+ pages)CRITICAL — the gutter prevents text from disappearing into the spine. Wider for thicker books.
CANVA

🎨 Canva Interior Creation Workflow

StepActionDetails
1Create custom sizeCanva → Create a Design → Custom Size. Enter: 6×9 inches OR 8.5×11 inches (no bleed adjustment needed since you're no-bleed).
2Set up ruler guidesFile → Show Rulers. Drag guides to create margins: 0.5" from top, bottom, outside edge. 0.75" from inside/gutter edge. ALL content stays inside these guides.
3Design first page typeBuild your first interior page (e.g., a lined journal page). Use lines, shapes, text boxes. Keep everything within margin guides.
4Duplicate to fill bookRight-click page → Duplicate Page. Repeat until you have the correct page count. For alternating layouts, duplicate different page types in sequence.
5Add front matterPage 1: Title page (book title, pen name). Page 2: Copyright page. Page 3: Table of contents or "This book belongs to" page. Page 4: Introduction or "How to use this book".
6Export as PDFShare → Download → PDF Print → Flatten PDF ☑️ → Download. This creates a 300 DPI print-ready PDF that KDP accepts.
Canva Free vs. Pro: Canva Free works for all 8 books. The only limitation: Free limits you to ~100 pages per design. For 120+ page books, split into two Canva files (front matter + first half, second half) and merge PDFs using a free tool like iLovePDF. Canva Pro ($13/mo) removes this limit — consider upgrading AFTER your first book earns revenue, not before.
KDP's reskinning rule (2025+): Amazon now rejects books that are identical interiors with only different covers. Each book MUST have unique interior content — different page layouts, different prompts, different headers. You cannot publish the same lined journal 5 times with different covers. All 8 of your books have genuinely different interiors by design, so this isn't a risk for you — but never try to shortcut by reusing an interior across multiple listings.

EXPORT

📤 PDF Export Settings & Quality Checklist

SettingValueWhy
File formatPDF PrintNOT PDF Standard — Print gives you 300 DPI resolution.
Flatten PDF✅ CheckedFlattening embeds all fonts and images. Prevents rendering issues on KDP's printers.
Color profileCanva default (sRGB)For B&W interiors, color profile doesn't matter — everything prints in greyscale. Don't overthink this.
Crop marks / bleed❌ UncheckedYou're no-bleed. Don't add crop marks or bleed. Leave both unchecked.
Pre-upload quality checklist:
CheckWhat to Verify
☐ Page countOpen PDF in any reader. Verify correct number of pages. Must be even number (KDP prints double-sided).
☐ MarginsZoom to 100%. Verify all text/lines are inside margins. Nothing closer than 0.5" to any edge, 0.625"–0.75" from inside/gutter edge.
☐ Text readabilityAll text at 10pt minimum. Headers 14–18pt. Line spacing 1.2–1.5. Serif font for body text (Georgia, Garamond). Sans-serif for headers (Arial, Outfit).
☐ No bleed contentNo lines, boxes, or images touching the page edge. White border visible on all four sides of every page.
☐ Front matter presentTitle page → Copyright → Belongs To / Profile → How to Use → then content begins.
☐ Consistent stylingSame fonts, line weights, header styles across all pages. Headers aligned. Tables uniform.
☐ KDP PreviewerAfter uploading to KDP, ALWAYS use the Launch Previewer to check every page. It flags margin violations and image issues.
